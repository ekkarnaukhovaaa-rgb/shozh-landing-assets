# ШОЖ — картинки для лендингов

Публичное хранилище изображений для лендингов Школы Осознанной Женственности
(фото, аватары отзывов и т.п.), чтобы вставлять их в лендинги по постоянным
ссылкам `raw.githubusercontent.com`.

Сюда кладём ТОЛЬКО картинки для публичных лендингов (они и так видны на сайте).
НЕ кладём: документы, данные, ключи, что-либо конфиденциальное.

Картинки — в папке `images/`.
